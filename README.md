# Lab-CSE-104: Practical Coursework

This repository contains all lab assignments and practical exercises for the **CSE-104** course. The goal is to document my learning progress and manage source code systematically.

## Repository Structure
The project is organized into 10 main directories, corresponding to 10 weeks of the course:
* `Lab01/` to `Lab04/`: Completed exercises and course materials.
* `Lab05/` to `Lab10/`: Upcoming lab sessions.

## Lab Content Summary

### Lab 01: Introduction to Programming and Problem Solving
* **Focus:** Course overview and essential problem-solving skills.
* **Exercises:** EIMONE, EIPOINT, EIUTHU, EIEVERYN, EIINCRDES, EIUTRIGLE.

### Lab 02: Review Programming Essentials
* **Focus:** Primitive types, built-in types, Math & Utilities classes, Debugging, and Version Control Systems.
* **Exercises:** EIUMADIS, EISNAIL, EIUFING, EIUBIRTH, EIUCUBES, EIUCUBES2, EIDISCOU, EIDISCOUNT2.

* **Exercises:** EIUCHRMS, EIUSALES, EIDISCOUNT3, EIAUCTION, EIMEMCARD, EIBANKFEE, EIBANKFEE2, EIUMARKUP.

* **Exercises:** EISALARY2, EITAX, EIGROSS, EIGROSS2, EIBANKRATE, EIDINRATE, EIAPP21222FQ1, EIAPP21222FQ2.
